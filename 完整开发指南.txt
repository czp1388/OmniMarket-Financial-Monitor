# 🎯 OmniMarket 完整开发流程指南

## 🚀 开始开发（每次开机后）：
### 方法1：桌面快捷方式（推荐）
1. 双击桌面上的 "开始开发.bat"
2. 复制生成的智能话术给DeepSeek
3. DeepSeek会提供具体命令

### 方法2：命令行
1. 打开 PowerShell
2. 输入：cd E:\OmniMarket-Financial-Monitor
3. 输入：.\终极开发.bat
4. 复制话术给DeepSeek

## 💾 结束开发（每次关闭聊天前）：
### 方法1：桌面快捷方式（推荐）
1. 双击桌面上的 "结束开发.bat"
2. 系统自动备份所有进度

### 方法2：命令行
1. 输入：.\结束开发.bat
2. 系统自动备份

## 🔄 开发连续性保障：
- 开始脚本：自动修复问题 + 启动服务 + 生成话术
- 结束脚本：自动备份进度 + Git提交 + 创建备份快照
- 进度文件：记录所有开发状态
- 备份系统：确保代码永不丢失

## 📋 核心文件说明：
- 开始开发.bat (桌面) → 启动开发环境
- 结束开发.bat (桌面) → 备份并结束开发
- progress.txt → 开发进度记录
- backups\ → 备份文件目录

记住：开始 → 开发 → 结束 → 备份 → 重复
